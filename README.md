# Układ roboczy koparki

Model 3D części układu roboczego koparki, odwzorowujący mechanikę ramienia oraz kinematykę siłownika.

### Komponenty układu:
* **Ramię i łyżka:** Wyposażona w zęby mocowane sztywno za pomocą nitów.
* **Układ napędowy:** Cylinder oraz tłok siłownika.
* **Mechanizm dźwigni:** Wahacze oraz przenośniki momentu (w plikach oznaczone jako „pazury”).

### Mechanika złożenia:
* **Połączenia sworzniowe i nitowe:** Zrealizowane przy użyciu więzów współosiowości oraz funkcji *touch*.
* **Kinematyka siłownika:** Współpraca elementów zdefiniowana przez współosiowość oraz funkcjonalne ograniczenia odległościowe.
