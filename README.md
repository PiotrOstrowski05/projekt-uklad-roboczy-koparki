# [cite_start]Układ roboczy koparki – Projekt CAD [cite: 1]

[cite_start]Model 3D części układu roboczego koparki, odwzorowujący mechanikę ramienia oraz kinematykę siłownika[cite: 3, 19].

### [cite_start]Komponenty układu: [cite: 7, 8, 9, 10, 11, 12, 13]
* [cite_start]**Ramię i łyżka:** Wyposażona w zęby mocowane sztywno za pomocą nitów[cite: 12, 13, 15].
* [cite_start]**Układ napędowy:** Cylinder oraz tłok siłownika[cite: 7, 11].
* [cite_start]**Mechanizm dźwigni:** Wahacze oraz przenośniki momentu, w plikach oznaczone jako „pazury”[cite: 8, 9, 17].

### Mechanika złożenia:
* [cite_start]**Połączenia sworzniowe i nitowe:** Zrealizowane przy użyciu więzów współosiowości oraz funkcji *touch*[cite: 16, 18].
* [cite_start]**Kinematyka siłownika:** Współpraca elementów zdefiniowana przez współosiowość oraz funkcjonalne ograniczenia odległościowe[cite: 19].
